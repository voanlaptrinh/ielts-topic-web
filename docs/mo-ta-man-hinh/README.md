# Tài liệu mô tả màn hình IELTS Focus

Thư mục này dùng để mô tả chức năng từng màn hình trước khi thiết kế lại giao diện.

Các file:

- `01-public.md`: Trang dành cho khách chưa đăng nhập.
- `02-auth.md`: Đăng nhập, đăng ký và menu tài khoản.
- `03-dashboard-hoc-vien.md`: Dashboard, tài khoản và lịch sử học viên.
- `04-noi-dung-hoc.md`: Chủ đề, từ vựng, flashcards, quiz và từ điển.
- `05-luyen-bai.md`: Reading, Listening, Writing, Speaking và lộ trình học.
- `06-admin.md`: Toàn bộ màn hình quản trị.

Khi cung cấp ảnh giao diện, nên đặt tên theo màn hình tương ứng, ví dụ:

- `home-guest-desktop.png`
- `home-guest-mobile.png`
- `dashboard-user.png`
- `admin-topic-list.png`
